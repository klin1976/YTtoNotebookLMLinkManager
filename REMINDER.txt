================================================================================
📝 對話紀錄更新提醒
================================================================================

⚠️ 重要提醒：在關閉專案前，請記得更新對話紀錄！

================================================================================
🔹 方法 1：直接告訴 Agent
================================================================================
在 Antigravity Agent 對話框中輸入以下任一指令：

  ✅ 「更新對話紀錄」
  ✅ 「我要關閉專案了」
  ✅ 「推送對話紀錄到 GitHub」

Agent 會自動更新 ConversationRecord.txt 並詢問是否推送到 GitHub。

================================================================================
🔹 方法 2：使用 VS Code Task
================================================================================
按下快捷鍵執行任務：

  1. 按 Ctrl+Shift+P 開啟命令面板
  2. 輸入 "Tasks: Run Task"
  3. 選擇「📝 更新對話紀錄提醒」

或者使用快捷鍵 Ctrl+Shift+B 執行預設建置任務。
